Rock Paper Scissors Game

This is a simple Rock-Paper-Scissors game implemented in Java using basic programming concepts like conditionals, user input, and random number generation.

>Features
1) Takes user input for Rock (0), Paper (1), or Scissors (2)
2) Generates a random choice for the computer
3) Compares both choices and determines the winner
4) Displays both numeric and readable (text) choices
5) Handles tie, win, and lose conditions

🛠️ Technologies Used
1) Java
2) Scanner class for user input
3) Random class for computer-generated moves

🎮 How It Works
1) The user enters a number:
0 → Rock
1 → Paper
2 → Scissors

2) The computer randomly selects a number between 0 and 2.
3) The program compares both inputs:
i)Same values → Tie
ii)Winning conditions → Human wins
iii)Otherwise → Computer wins
4) The program prints both choices in numeric and readable format.
